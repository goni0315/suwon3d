<가로수라이브러리 선택>


① 가로수 구간설정
    - [구간선택]버튼을 클릭하여 지도 화면에서 가로수구간을 마우스 왼쪽버튼으로 설정합니다.
② 가로수 선택 및 설정
    - 가로수를 선택한 후 가로수 옵션을 설정한 후 [시설물 삽입]버튼을 클릭하면 지도화면에
      선택된 가로수가 표현됩니다.
③ 미리보기
    - 가로수선택 ⇨ 가로수를 선택하면 해당 가로수가 미리보기에서 나타납니다.

 


<사용자 라이브러리 추가>


① 가로수구간설정
    - [구간선택]버튼을 클릭하여 지도 화면에서 가로수구간을 마우스 왼쪽버튼으로 설정합니다.
② 사용자 라이브러리 추가
    - [찾기]버튼으로 가로수 라이브러리를 추가하고 가로수 옵션을 설정 한 후에 [시설물삽입]
      버튼으로 선택합니다.
③ 미리보기
    - [찾기]버튼 ⇨ 가로수를 선택하면 해당 가로수가 미리보기에서 나타납니다.
