<시설물 라이브러리 선택>


① 시설물 라이브러리 선택
    - 시설물을 선택한 후 [시설물삽입]버튼을 클릭합니다.
    - 시설물 위치지정 알림창이 뜬 후 지도 화면에 입지할 지점에 클릭을 하면 선택한 데이터를
      지도화면에 불러옵니다.
    - [취소] 버튼을 클릭하면 마우스 포인터가 손 모양으로 변환됩니다.
    - [분석결과삭제] 버튼을 클릭하면 해당 시설물이 지도에서 사라집니다.
② 미리보기
    - 시설분류 ⇨ 시설물 ⇨ 시설물선택하면 해당 시설물 라이브러리가 미리보기에서 나타납니다.

 


<사용자 라이브러리 추가>


① 사용자 라이브러리 추가
    - [찾기] 버튼으로 추가할 시설물(.3DS파일)을 선택한 후 [시설물 삽입] 버튼을 클릭합니다.
    - 시설물 위치지정 알림창이 뜬 후 지도화면에 입지할 지점에 클릭을 하면 선택한 데이터를
      지도 화면에 불러옵니다.
② 미리보기 영역
    - [찾기]버튼 ⇨ 시설물을 선택하면 해당 시설물이 미리보기에서 나타납니다.
