안전길 등록

 

① 안전길 선택 영역

 - 상단 메인메뉴에서 안전길을 선택하면 보안등 위치 정보 조회 화면이 나타납니다.
② 양식화면 영역 

  - [위치설정] : 지도화면에서 위치 설정 합니다.

  - [이동] : 보안등 위치설정 후 다른 곳 이동 시 사용합니다.

  - [선택결과삭제] : 잘못 설정된 보안등 위치를 삭제합니다.

  - [초기화] : 해당 버튼 클릭 시 좌표 목록과 지도상에 표시된 위치가 모두 삭제됩니다.

  - 좌표가 지정 완료되면 [엑셀추출] 버튼 클릭하여 저장합니다.

  - 저장된 엑셀자료 양식에 안전길 범례를 참조하여 보안등(CCTV), 높이, 조도분포도를

    입력하여 저장합니다.

 ③ 안전길 시뮬레이션 실행 영역

  - [찾아보기]버튼을 클릭하여 자신이 작성한 엑셀파일을 선택하고 작성파일 올리기

    [엑셀등록] 버튼을 클릭합니다.

  - 자신이 작성한 양식에 맞춰 해당 위치를 찾아서 3차원화면에 아이콘 표시가 됩니다.
