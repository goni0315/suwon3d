내주제도 지번, 도로명

 

① 양식선택

  - [지번]/[도로명] 중에 하나를 선택하고 [양식내려받기] 버튼을 클릭합니다.
 ② 양식화면

  - 지번Sheet를 사용할 경우 내려받은 엑셀양식에 명칭, 소재지(구,동),
    산구분(토지,임야), 본번,부번, 색상, 수치를 입력하여
    저장위치를 확인하여 저장합니다.
  - 도로명Sheet를 사용할 경우 내려받은 엑셀양식에 명칭, 소재지(구, 도로명)
    건물번호(본번,부번), 색상, 수치를 입력하여 저장위치를
    확인하여 저장합니다.

 ③ 내주제도 실행

  - [찾아보기]버튼을 클릭하여 자신이 작성한 엑셀파일을 선택하고
    [작성파일올리기] 버튼을 클릭합니다.
  - 자신이 작성한 양식에 맞춰 해당 위치를 찾아서 3차원화면에 아이콘 표시가 됩니다.

 


내주제도 좌표

 

① 좌표선택

  - [좌표]를 선택하면 중간에 내주제도 좌표 위치 정보 조회 화면이 나타납니다.
 ② 양식화면

  - [위치설정] : 지도화면에서 좌표 위치를 설정합니다.
  - [이동] : 좌표 위치 설정 후 현재 보고 있는 지도 화면 외의 좌표 위치 설정을 위한
    지도 이동버튼입니다.
  - [선택결과삭제] : 잘못 설정된 좌표 위치를 삭제(체크박스 체크 후 클릭 시
    해당 좌표 삭제 됨) 합니다.
  - [초기화] : 해당 버튼 클릭 시 좌표 목록과 지도상에 표시된 위치가 모두 삭제됩니다.
  - 좌표가 지정 완료되면 【엑셀추출】버튼 클릭하여 저장합니다.
  - 저장된 엑셀자료 양식에 내주제도 범례를 참조하여 명칭, 색상, 수치를 입력하여 저장합니다.

 ③ 내주제도 실행

  - [찾아보기]버튼을 클릭하여 자신이 작성한 엑셀파일을 선택하고
    [작성파일올리기] 버튼을 클릭합니다.
  - 자신이 작성한 양식에 맞춰 해당 위치를 찾아서 3차원화면에 아이콘 표시가 됩니다.
