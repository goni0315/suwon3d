<조망권 분석>

 

① 조망지점설정
    - [조망지점 설정] 버튼을 클릭한 후 지도화면에 지점을 선택시 알림창이 생깁니다.
    - 지도화면에 기준점 및 방향점2개 이상 표시되면 [적용] 버튼을 클릭합니다.
    - [분석결과삭제] 버튼을 클릭하면 방향점 리스트가 삭제되며, 지도화면에서
       조망지점이 사라집니다.
② 방향점 리스트
    - 높이값과 시뮬레이션 속도를 입력하여 조절가능
    - 조망지점 설정 후 [적용] 버튼을 클릭하면 지도화면에서 기준점으로 이동한 후
      각 방향점들을 향해 순차적으로 바라보게 됩니다.

 


<조망지점관리>


① 가시지점입력
    - [가시지점입력] 버튼을 클릭한 후 지도화면에 지점을 선택하면 대상지가 표시됩니다.
    - 지점명을 입력하고 바로 밑에 있는 [가시지점명입력] 버튼을 클릭한 후 지도화면에 지점을
      선택하면 지점명이 표시됩니다.
    - [분석결과삭제] 버튼을 클릭하면 조망점 리스트가 삭제되며, 지도화면에서 가시지점이
      사라집니다.
② 조망점 리스트
    - 가시지점 설정 후 [적용] 버튼을 클릭하면 지도화면에서 지점명으로 이동한 후
      대상지를 향해 바라봄니다.
    - 현재위치의 X, Y, Z 좌표 값이 반영됩니다.

 


<조망축관리>


① 조망지점 설정
    - [조망지점 설정] 버튼을 클릭한 후 지도화면에 시작점과 끝점을 선택합니다.
    - 지점명에 입력을 하고 [등록] 버튼을 클릭합니다.
    - [분석결과삭제] 버튼을 클릭하면 지점명이 삭제되며, 지도화면에서 조망축이 사라집니다.
    - 관측시점란에 [저장]버튼을 클릭하면 등록된 조망축이 저장됩니다.
    - 관측시점란에 주요지점 리스트를 선택하면 [불러오기]버튼이 생성됩니다.
    - 관측시점란에 [불러오기]버튼을 클릭하면 저장된 지점명 리스트를 불러옵니다.
② 관찰높이 설정
    - 관찰높이 수치(숫자)를 입력하고 지점명을 선택하면 입력한 수치의 높이에서 관찰합니다.
③ 지점명목록
    - 조망지점 설정 후 [등록] 버튼을 클릭하면 지도화면에서 조망축이 나타나며 등록한
      지점명이 지점명 리스트에 생깁니다.
