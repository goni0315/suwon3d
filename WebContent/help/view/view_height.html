영역선택을 이용하여 시설물들의 평균 높이를 구합니다.

 

① 영역(범위)선택
    - [객체선택] 버튼을 클릭한 후 지도화면에서 객체를 지정합니다.
    - [영역선택] 버튼을 클릭한 후 지도화면에서 마우스 왼쪽버튼으로 영역범위를 지정하고
      마우스 오른쪽버튼을 클릭합니다.
    - [원형선택] 버튼을 클릭한 후 지도화면에서 마우스 왼쪽버튼으로 영역범위를 지정하고
      마우스 오른쪽버튼을 클릭합니다.
    - [분석결과삭제] 버튼을 클릭하면 지도화면에서 선택된 영역표시가 사라집니다.
② 분석대상
    - 층높이에 값을 입력하여 평균높이 결과 값을 조절가능합니다.
    - [적용]버튼을 클릭하면 각 건물의 레이어명, 고유번호, 높이가 [분석대상] 칸에 표현됩니다.
    - [평균높이 결과 값] 칸에 각 건물들의 평균높이 결과 값이 산출되며 3차원화면에
      각 건물의 높이가 표시됩니다.
