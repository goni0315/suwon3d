<반경검색>


① 반경검색
    - [기준위치선택] 버튼을 클릭하여 지도화면에 검색할 위치를 클릭합니다.
    - 반경검색옵션을 입력 한 후 검색버튼을 클릭합니다.
    ※ 건물분류 : 특정 건물들 검색의 가독성
    ex) 건물분류 : 공공기관 ⇨ 수원시청, 수원시 견인사무소
② 검색결과
    - 검색결과 반경 안에 있는 건물들의 건물명, 도로명 주소를 조회합니다.
③ 대장조회
    - 검색결과에서 [대장조회] 버튼을 클릭하면 건축물대장, 토지이용계획, 개별공시지가 등
      조회 가능합니다.

 


<영역검색>


① 영역검색
    - [영역그리기] 버튼을 클릭하여 지도화면에 검색할 영역 3지점이상을 클릭하여 지정합니다.     - 검색 옵션을 입력 한 후 검색 버튼을 클릭합니다.
② 검색결과
    - 검색결과 반경 안에 있는 건물들의 건물명, 도로명 주소를 조회합니다.
③ 대장조회
    - 검색결과에서 [대장조회] 버튼을 클릭하면 건축물대장, 토지이용계획, 개별공시지가 등
      조회 가능합니다.
