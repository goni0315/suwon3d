분석 도구

 

① 거리 측정 : 지도 상의 거리(길이)를 측정 할 수 있습니다.
    - 3차원화면에서 거리를 측정하고자 하는 두 점 이상의 포인트를 입력합니다.
    - 거리측정을 중지하기 위해서는 마지막 지점에서 마우스 오른쪽 버튼을 클릭합니다.

② 높이 측정 : 지도 상에 보여지는 지형, 건물 시설물 등을 선택하시면
                      해당 선택 객체의 높이 값을 나타내 줍니다.

③ 면적 측정 : 지도 상에 3개의 점 이상을 클릭하시면 해당 면적을 보실 수 있습니다.
    - 면적 측정을 중지하기 위해서는 마지막 지점에서 마우스 오른쪽 버튼을 클릭합니다.

④ 부피 측정 : 높이 측정과 동일하게 부피를 측정하실 건물을 선택하시면
                      부피를 확인 하 실 수 있습니다.

⑤ 단면 분석 : 특정 구간 선택 시 구간 사이의 단면도를 확인 할 수 있습니다.
    - [지점입력] 클릭 ⇨ [목표점입력] 클릭 ⇨ [분석실행] 클릭 순으로 실행합니다.

⑥ 경사 분석 : 경사분석과 경사향분석을 할 수 있는 기능입니다.
    - 경사 분석 : 경사 분석을 하고자하는 영역을 [영역선택] 버튼으로 선택한 후
                      [경사분석실행]버튼으로 실행합니다.
    - 경사향 분석 : 경사향 분석을 하고자하는 영역을 [영역선택] 버튼으로 선택한 후
                      [경사분석실행]버튼으로 실행합니다.



내비게이션

 

① 경로기능 : 지도 상에 경로 설정, 경로저장, 경로 불러오기, 경로삭제를 수행합니다.

② 비행기 및 자동차 모드 : 경로 이동 시 하늘과 땅에 이동 할지 결정합니다.

③ 속도 : 경로 이동 시의 속도를 결정합니다.

④ 화면시점 : 실행 경로에서 바라볼 방향을 설정합니다.

⑤ 고도 : 비행기 모드의 경우 고도를 설정합니다.

⑥ 녹화기능 : 녹화 버튼 클릭 시 반디캠의 다운로드 페이지로 이동합니다.

⑦ 실행제어 : 경로 이동 시 재생, 일시정지, 정지의 기능이 있습니다.

⑧ 도구모음 : 기본 툴바의 기능이 네비게이션에 연결되어있습니다.



기본툴바

 

① 날씨 : 수원시 전체의 날씨를 지도화면에 표현합니다.

② 정보 : 마우스가 정보조회 모드로 전환되며 이 상태에서 건물, 지번, 지하시설물을
                      선택하여 해당 상세정보를 조회합니다.

③ 1인칭시점 : 지도화면을 보고 있는 시점을 1인칭 시점으로 변경됩니다.


④ 3인칭시점 : 지도화면을 보고 있는 시점이 3인칭 시점으로 변경됩니다.


⑤ 손이동 : 마우스 모드를 지도 이동모드로 변경하여 이동됩니다.

⑥ 출력 : 현재보고 있는 3차원화면을 프린트를 이용하여 출력합니다.

⑦ 저장 : 현재보고 있는 3차원화면을 이미지 파일로 저장합니다.

⑧ 전체화면 : 지도화면을 전체로 볼 수 있는 화면이 됩니다.

⑨ 초기화 : 지도 화면 상에 실행되었던 기능들을 초기화합니다.
